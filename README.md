# UniMap

Indoor maps pentru universități (Craiova → prima).
**Stack:** SwiftUI, MapKit, JSON pentru hărți, editor intern.

## Rulare
1. Deschide `UniMap.xcodeproj`
2. Schema: UniMap
3. Rulează (⌘R)

## Structură
- App/ — entry
- Features/ — ecrane
- Shared/ — componente comune
- Data/Maps — hărți JSON
